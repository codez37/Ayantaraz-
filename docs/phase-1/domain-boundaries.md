# Domain Boundaries

## Inside Domain
- Educational content publication (articles, videos, mini-books)
- Course-based learning with sample video previews
- Consultation lead generation and tracking
- Manual/offline payment for course access
- Phone-based authentication via Kavenegar OTP
- Admin content management with lifecycle control
- Constrained knowledge-base chatbot
- Audit logging for compliance

## Outside Domain
- Enterprise financial calculations
- Full legal document automation
- Online payment processing
- Heavy user profiling or analytics tracking
- Unsupervised AI/ML chatbot behavior
- Social media or community features
- Real-time financial data feeds
- Integration with accounting software (future possibility, not Phase 1-8)
- Mobile native application
- Multi-tenant SaaS architecture

## Boundary Rules
- Any feature crossing outside domain requires explicit User Owner approval
- No feature may be added that violates the data minimization rule
- No feature may be added that requires a payment gateway
- No feature may be added that gives the chatbot unconstrained authority
