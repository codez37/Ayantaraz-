# Out of Scope

The following items are explicitly excluded from this project:

1. **Mobile application** (iOS/Android native)
2. **Digital wallet** or credit system
3. **Payment gateway integration** (Shaparak, Zarinpal, etc.)
4. **Heavy CRM system** (full customer relationship management)
5. **Enterprise ERP** integration
6. **Microservice architecture** (unless explicitly justified by bottleneck)
7. **Recommendation engine** (AI-based content suggestions)
8. **Unconstrained AI chatbot** (no LLM or generative AI)
9. **Social network features** (comments, likes, sharing, user posts)
10. **Full enterprise accounting software**
11. **Complex tax automation** beyond basic educational content
12. **Real-time financial data** or market feeds
13. **Email marketing automation**
14. **User analytics** (beyond basic page views for admin)
15. **Multi-language support** (Persian only)
16. **API for third-party developers**
17. **Progressive Web App (PWA)**
18. **Offline mode**
19. **User-to-user messaging**
20. **Advanced search with Elasticsearch** (basic DB search is sufficient)

Any request to add these features must be escalated to User Owner for explicit approval.
