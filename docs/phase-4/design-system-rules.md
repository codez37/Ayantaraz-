# Design System Rules

## Typography
- Font: Vazirmatn (Persian)
- Weights: 400 (regular), 700 (bold)
- Scale: 14px (body), 16px (large body), 20px (h3), 24px (h2), 32px (h1)
- Line height: 1.6 (body), 1.3 (headings)
- RTL: always

## Colors (Proposed)
- Primary: Blue (#2563EB) — trust, authority
- Secondary: Teal (#0D9488) — CTA, action
- Success: Green (#16A34A)
- Warning: Yellow (#D97706)
- Error: Red (#DC2626)
- Neutral: Gray 50-900 for structure
- Background: White (#FFFFFF) for cards, Gray 50 (#F9FAFB) for page

## Spacing
- 4px base unit
- Content max-width: 1280px
- Card padding: 24px
- Section spacing: 64px (desktop), 32px (mobile)

## Buttons
- Primary: filled, primary color, white text
- Secondary: outlined, primary color border
- Danger: red filled or red outlined
- Ghost: no background, text only
- Size: sm (32px), md (40px), lg (48px)
- Border radius: 8px

## Inputs
- Height: 40px (text), 120px min (textarea)
- Border: 1px solid gray-300
- Focus: 2px primary color ring
- Error: red border + error text below
- Label: above input, 14px, bold

## Cards
- Background: white
- Border: 1px solid gray-200
- Border radius: 12px
- Shadow: subtle (sm)
- Padding: 24px

## Tables
- Header: gray-50 background, bold
- Row: white, alternate gray-50
- Hover: gray-100
- Border: bottom line only
- Responsive: horizontal scroll on mobile

## Badges / Status
- Published/Confirmed: green bg, green text
- Draft/Pending: yellow bg, yellow text
- Archived/Rejected: gray bg, gray text

## Icons
- Use Lucide React for consistency
- 16px for inline, 24px for buttons, 32px for section icons
