# Responsive Behavior Rules

## Breakpoints
- Mobile: 0-639px
- Tablet: 640-1023px
- Desktop: 1024px+

## Layout Adaptation

### Header
- Mobile: hamburger menu, slide-out nav
- Tablet: condensed nav (icons + labels)
- Desktop: full horizontal nav

### Homepage Hero
- Mobile: stacked, CTA buttons full width
- Desktop: side-by-side, CTA buttons inline

### Content Grid
- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3 columns (articles), 4 columns (courses)

### Tables
- Mobile: card layout (label + value per row)
- Tablet: condensed table with horizontal scroll
- Desktop: full table with all columns

### Forms
- Mobile: stacked, full width
- Desktop: max-width 480px (auth), 640px (consultation)

### Admin Panel
- Mobile: sidebar collapsed, overlay menu
- Desktop: persistent sidebar, full-width content

### Chatbot Widget
- Mobile: bottom sheet (60% height)
- Desktop: fixed position, 380px width

## Touch Targets
- Minimum: 44x44px for all interactive elements
- Buttons: minimum height 40px
- Links: minimum 16px with adequate padding
