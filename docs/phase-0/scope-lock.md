# Scope Lock

## In Scope
- Accounting services website
- Tax consultation lead generation
- Educational content (articles, videos, mini-books)
- Phone-based registration/login with Kavenegar OTP
- Admin content management panel
- Consultation request workflow
- Course sales without payment gateway
- Constrained knowledge-based chatbot
- Content lifecycle management
- Audit logging

## Out of Scope
- Mobile application
- Digital wallet
- Payment gateway integration
- Heavy CRM system
- Full ERP
- Microservice architecture
- Recommendation engine
- Unconstrained AI/chatbot
- Social network features
- Full enterprise accounting
- Complex tax automation outside domain

## Priority Order (Phase 1-8)
1. Authentication and user management
2. Content management (articles, videos, mini-books)
3. Courses and course videos
4. Manual payment workflow
5. Consultation request flow
6. Constrained chatbot
7. Frontend (Next.js)
8. Production hardening and deployment
