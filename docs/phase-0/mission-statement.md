# Mission Statement

## Core Mission
Build a production-grade Persian website for accounting services, tax consultation, educational content, and content management, with phone-based authentication, manual payment workflow, constrained chatbot, and admin control surface.

## Primary Objectives
- Attract and convert clients for accounting/tax services
- Generate qualified consultation leads
- Sell educational courses via manual payment
- Publish trusted educational content
- Maintain strict data minimization
- Ensure full auditability and rollback capability

## Secondary Objectives
- Build user trust through transparent workflows
- Provide constrained, safe tax chatbot
- Enable admin content management with full lifecycle control
- Prepare for real production deployment

## Success Definition
The system is successful when it can survive:
- Real users with real auth flows
- Real admins making real content/payment decisions
- Real content changes with review lifecycle
- Real operational mistakes with rollback paths
- Real deployment with monitoring and backup
- Harsh technical review of security, audit, and domain correctness
