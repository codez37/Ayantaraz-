# Approved Answers

## Structure
Each approved answer has:
- id
- questionPattern: keyword/phrase pattern
- answer: controlled response text
- category: grouping
- riskLevel: low/medium/high/forbidden
- status: active/archived
- sourceType: knowledge_base / faq / article

## Lifecycle
draft → active → archived

## Management
- Created by admin/content_manager
- Edited by admin/content_manager
- Archived (never deleted)
- Audit logged on every change

## Content Requirements
- Answer must be factual
- Answer must cite source category
- Answer must NOT promise specific tax outcomes
- Medium risk answers must include disclaimer
