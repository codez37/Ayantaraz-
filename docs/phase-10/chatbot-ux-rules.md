# Chatbot UX Rules

## UI Requirements
- Simple chat interface
- Placeholder: "سوال خود را بپرسید..."
- Bot responses in Persian, concise
- "ارتباط با مشاور" button visible
- Disclaimer shown on first load
- Typing indicator while processing

## Disclaimer (shown once per session)
"این چت‌بات راهنمای آموزشی است و جایگزین مشاوره تخصصی مالیاتی نمی‌شود. برای تصمیم‌گیری دقیق با کارشناسان ما مشورت کنید."

## User Understanding
The user must understand:
- Chatbot is not a legal authority
- Answers are educational and controlled
- Sensitive questions require human consultant

## Safety Messages
- Low risk: no extra message
- Medium risk: "این پاسخ عمومی است و برای تصمیم نهایی نیاز به بررسی تخصصی دارد."
- High risk: "برای این موضوع، مشاوره انسانی توصیه می‌شود."
- Forbidden: "این سوال خارج از حوزه پاسخ‌های تاییدشده است."
- Fallback: "متأسفم، نمی‌توانم به این سوال پاسخ دهم. لطفاً برای راهنمایی دقیق با کارشناسان ما تماس بگیرید."
