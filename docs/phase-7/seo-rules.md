# SEO Rules

## Required for All Published Content

### Slug
- Must be unique across all content
- Persian or English, URL-safe
- Example: `maliyat-bar-arz-afzudeh`

### Title
- 30-60 characters recommended
- Include primary keyword
- Unique per page

### Meta Description (recommended)
- 120-158 characters
- Include call-to-action
- Include primary keyword

## Heading Structure

```
h1: Title (auto-generated from content title)
h2: Main sections
h3: Subsections
```

## URL Structure

```
/articles/{slug}
/videos/{slug}
/minibooks/{slug}
/faq (collection)
/page/{slug} (static pages)
```

## Internal Linking

- Related articles at bottom of each article
- Category pages link to all articles in category
- Articles can link to courses/consultation pages

## Canonical
- Each content piece has canonical URL = its own URL
- No duplicate content issues expected
