# Assignment Policy

## Who Can Assign
- Admin → assign to self or any operator
- Operator → assign only to self (claim)

## Assignment Rules
1. Every request may be assigned to at most one person at a time
2. Assignment is logged in audit
3. Re-assignment overwrites previous assignee (with audit)
4. Unassigned requests visible to all operators
5. Assigned requests visible to assignee + admin

## Auto-Assignment (Future)
Not implemented. All assignment is manual in this phase.

## Dashboard Views
- Unassigned pending requests (all operators see)
- My assigned requests (per-operator)
- All requests (admin)
