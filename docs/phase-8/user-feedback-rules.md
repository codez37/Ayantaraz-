# User Feedback Rules

## After Successful Submission
- عنوان: "درخواست شما با موفقیت ثبت شد"
- کد پیگیری: `#1234`
- متن: "کارشناسان ما در اسرع وقت با شما تماس خواهند گرفت"
- لینک: "پیگیری درخواست‌های من" → /dashboard/requests

## After Error
- خطای واضح و فارسی
- دلیل قابل فهم (شماره تلفن نامعتبر، متن کوتاه، محدودیت تعداد)
- دکمه تلاش مجدد
- نمایش ندادن جزئیات داخلی/فنی

## User Dashboard (User-Facing)
| فیلد | توضیح |
|------|-------|
| شماره پیگیری | #1234 |
| موضوع | حسابداری / مالیات / عمومی |
| وضعیت | به فارسی (در انتظار بررسی / تماس گرفته شد / ...) |
| تاریخ ثبت | 1403/01/15 |
| آخرین بروزرسانی | 1403/01/20 |
| پیام نهایی | اگر تکمیل شده، خلاصه نتیجه |

## Status Labels (User-Facing Persian)
| Status | Persian Label |
|--------|---------------|
| pending | در انتظار بررسی |
| contacted | تماس گرفته شد |
| scheduled | زمان‌بندی شده |
| completed | تکمیل شده |
| canceled | لغو شده |
| no_response | بدون پاسخ |
| rejected | رد شده |
